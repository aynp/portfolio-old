# Compile & Run C++

```json
{
  "cmd": [
    "g++ ${file} -std=c++20 -o a.out && ./a.out > output.txt < ./input.txt"
  ],
  "file_regex": "^(..[^:]*):([0-9]+):?([0-9]+)?:? (.*)$",
  "shell": true,
  "working_dir": "${file_path}",
  "selector": "source.cpp, source.c++"
}
```
